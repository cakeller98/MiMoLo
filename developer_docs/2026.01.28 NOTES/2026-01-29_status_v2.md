# 2026-01-29 Status Update

## Packaging utilities
- `release-pack-agent` now supports:
  - Release flags (mutually exclusive): --major | --minor | --patch
  - Prerelease flags (mutually exclusive): --alpha | --beta | --rc
  - You can combine one release + one prerelease flag
- Combined release+prerelease uses semver premajor/preminor/prepatch:
  - Example: 1.0.1-beta.0 + --major --beta => 2.0.0-beta.0
- Uses semver standard behavior for prereleases:
  - alpha/beta/rc with dotted suffix (e.g., alpha.0, beta.0)

## Location & use
- Tool lives in `mimolo/utils/src/release-pack-agent.ts`
- Recommended run:
  - `npx tsx src/release-pack-agent.ts --source "../agents/agent_template" --beta`
- `--out` is resolved relative to `--source` when relative. Default is `../repository`.

## Notes
- `build-manifest.toml` is the source manifest.
- `manifest.json` and `payload_hashes.json` are generated during packaging.
